# AstrBot WOT Plugin

AstrBot 插件，用于查询《坦克世界》玩家效率

## 目录说明

- `main.py`: AstrBot 插件注册与命令入口。
- `src/application`: 应用层，负责编排命令处理、查询流程和报表流程。
- `src/domain`: 领域模型与领域枚举。
- `src/infrastructure`: 基础设施层，包含网络客户端、解析器、仓储与网关实现。
- `src/infrastructure/api_clients`: API客户端实现。
- `src/settings`: 配置常量与消息模板。
- `src/tasks`: 定时任务入口（例如坦克数据同步调度）。
- `resources/static`: 模板、字体和静态数据。
- `数据存储`: 绑定数据和坦克数据存储在 `data/plugin_data/astrbot_plugin_wot/data/` 目录下。
- `运行时产物`: 生成的HTML和PNG报表存储在 `data/temp/astrbot_plugin_wot/report/` 目录下。

## 命令说明

### 基础命令

- **帮助** / **help**：显示所有可用命令及其说明
  - 示例：`/帮助` 或 `帮助` 或 `/help` 或 `help`
- **wot绑定**：绑定玩家游戏名称到当前QQ账号
  - 示例：`/wot绑定 玩家名称` 或 `wot绑定 玩家名称`
- **效率** / **盒子效率**：查询盒子页面基础效率数据（文本返回）
  - 示例：`/效率` 或 `/效率 玩家名称` 或 `效率` 或 `效率 玩家名称`

### 战绩报表

- **今日效率** / **今日战绩**：查询今日效率和战绩
  - 示例：`/今日效率` 或 `/今日效率 玩家名称` 或 `今日效率` 或 `今日效率 玩家名称`
- **昨日效率** / **昨日战绩**：查询昨日效率和战绩
  - 示例：`/昨日效率` 或 `/昨日效率 玩家名称` 或 `昨日效率` 或 `昨日效率 玩家名称`
- **两日效率** / **两日战绩**：查询两日效率和战绩
  - 示例：`/两日效率` 或 `/两日效率 玩家名称` 或 `两日效率` 或 `两日效率 玩家名称`
- **三日效率** / **三日战绩**：查询三日效率和战绩
  - 示例：`/三日效率` 或 `/三日效率 玩家名称` 或 `三日效率` 或 `三日效率 玩家名称`
- **百场效率** / **百场战绩**：查询百场效率和战绩
  - 示例：`/百场效率` 或 `/百场效率 玩家名称` 或 `百场效率` 或 `百场效率 玩家名称`

### 管理命令

- **同步坦克** / **更新坦克**：融合官网与 WotInspector 的坦克信息
  - 示例：`/同步坦克` 或 `同步坦克`

### 命令使用说明

所有命令都支持两种使用方式：
- **带唤醒词**：使用 `/` 前缀，例如 `/效率`
- **不带唤醒词**：直接输入命令，例如 `效率`

对于官方QQ机器人，需要先@机器人，然后再输入命令，例如：`@QQ机器人 效率` 或 `@QQ机器人 /效率`

## 环境要求

本插件使用 Html2Image 库生成战绩报表图片，需要在运行环境中安装浏览器（Chromium/Chrome）。

### Docker环境配置

如果使用AstrBot的Docker容器（基于Debian 13），需要进入容器安装Chromium浏览器：

1. **进入Docker容器**：
   ```bash
   docker exec -it <容器名称或ID> /bin/bash
   ```

2. **换源并安装Chromium**：
   ```bash
   sed -i 's|deb.debian.org|mirrors.aliyun.com|g' /etc/apt/sources.list.d/debian.sources && apt update && apt install -y chromium
   ```

安装完成后，插件即可正常生成战绩报表图片。
